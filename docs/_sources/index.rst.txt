Overview
========

.. image:: ./_static/img.png
   :align: center

Protomix is a Python package designed for the preprocessing of 1H NMR metabolomics data, facilitating efficient and accurate analysis of complex metabolomic datasets. This documentation offers step-by-step tutorials and a detailed manual to guide users through the installation and usage of Protomix.
